# 古文批量脚注-VSCode正则表达式+分段查找
## 使用支持正则表达式的文本编辑器（如 VS Code

1、处理正文中的脚注引用

- **查找内容**：`\[(\d+)\]`  （匹配所有 `[数字]`，如 `[1]`、`[2]`）

- **替换为**：`[^$1]`  （转换为 `[^1]`、`[^2]`）

- **适用范围**：仅针对正文部分（即注释之前的内容）
